1产品是什么

一个蹲在桌面上的像素 / 拟物萌宠，替你实时盯着 Claude Code、Codex、Cursor 等 20+ 个 AI 编码 Agent——它们在思考、打字、构建、报错、等你审批权限时，萌宠会用动画和气泡告诉你，让你不用一直盯着终端。

基于开源项目 clawd-on-desk（Electron，原形象是只像素螃蟹 Clawd，名字取自 Claude + claw 爪子）二次开发，我们要给它一个更朗朗上口、对中文路演观众更友好的名字与形象。

命名要戳中三件事：① 盯 / 守（产品核心动作）　② 萌宠陪伴感　③ 和 Claude / Code 有谐音钩子

2竞品命名参照系

产品命名逻辑给我们的启示
Clawd（原项目）Claude + claw（爪子），谐音梗谐音钩子是这条赛道的通用语
WorkBuddy（腾讯）Work + Buddy =「工作伙伴」Buddy/伙伴=陪伴感，国内已被验证好传播
Codex（OpenAI）Code + -ex 拉丁词尾「Code 家族」气质，国际范
Masko Code / Cloudling / Calico萌后缀 -ling / -o + watch 调性萌系小后缀好记好爱

3候选方案（按推荐排序）

排名中文主名英文代号拆解评价

🥈码宝Codo码=Code，宝=宝贝 / 萌宠国内最好叫、最像萌宠，一秒懂
🥉看看Kanon叠字「看看」=替你看着 Agent口语化、扣「盯 / 看」动作，亲切
为什么定「小哒 / Coda」：
  • Coda 是乐谱里的「收尾记号」——精准隐喻它陪你把 AI 写的活儿盯到收尾交付，立意比单纯「盯着」高一层；
  • Coda 谐音 Code，和 Codex / Claude 同属「code 家族」气质，好拼好读、国际范；
  • 中文「小哒」自带萌宠昵称感 +「哒」的轻快提示音，路演叫起来亲切上口；
  • 形象不被螃蟹 / 猫绑死，做成原创小精灵，配 3D 拟物质感更出片。

4形象设定

主体

原创圆滚滚小精灵，两只圆亮的大眼睛专注望着前方——又萌又机灵，象征它替你看着 AI Agent 把活干完。

质感

3D 拟物渲染，软胶 / 磨砂塑料玩偶质感 + 柔和光泽，正经产品 logo 级。

配色

暖琥珀橙 + 奶白，对齐作战中心暖米编辑风（accent #e0623a）。

用途

桌面 App 图标 + 路演主视觉 + 各动画状态的母版形象。
