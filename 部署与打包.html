1 推后端代码到 GitHub

已改好的文件：agent/server.py（支持 PORT 环境变量 + 访问令牌护栏）、render.yaml、agent/requirements.txt。密钥不在仓库里，部署时在 Render 控制台填。

2 在 Render 部署后端

后端零三方依赖、纯 Python 标准库，Render 免费层即可。

  1. 打开 dashboard.render.com → 注册/登录（可用 GitHub 账号）
  2. 右上 New + → Blueprint（蓝图，读仓库里的 render.yaml）
  3. 连接 GitHub → 选 diaojz/enter-key-hackathon 仓库 → Render 自动识别 render.yaml，建出名为 coda-agent 的 Web Service
  4. 填环境变量（Environment 标签，最关键的一步）：
    变量名填什么
    OPENAI_API_KEY主办方中转站那把 key（sk-DY7H…，控制台「API Key」标签）
    CODA_ACCESS_TOKEN自己编一串随机字符串，如 coda-2026-xxxx（护栏，防额度被刷）
    OPENAI_BASE_URLhttps://api.openai-next.com/v1（蓝图已预填，确认即可）
    CODA_LLM_MODELgpt-4o-mini 或 gpt-5（蓝图已预填）
    CODA_LLM_BACKENDopenai（蓝图已预填）
  5. 点 Deploy，等 1～2 分钟出绿色 Live
  6. 记下分配的域名，形如 https://coda-agent.onrender.com
验证：浏览器打开 https://你的域名/health，看到 {"ok": true} 即后端上线。
⚠ 免费层会休眠：15 分钟没人访问会睡，下次访问要等约 30 秒冷启动。路演前 5 分钟先打开一次 /health 唤醒它，避免评委点扫盘时卡在冷启动。
🔒 key 暴露提醒：内置 key 托管在公网，已加 CODA_ACCESS_TOKEN 护栏（请求要带 x-coda-token 头）。桌宠会自动带这个头，但 token 值要和下面步骤③填的一致。

3 桌宠填入云端地址 + 令牌

桌宠源码：/Users/diaoye/Desktop/coda-desktop。改造已做好，只需把两个常量改成真值。

  1. 打开 src/coda-eval.html，在文件顶部找到这两个常量（带中文注释，很显眼）： 
    const CODA_CLOUD_API   = 'https://coda-agent.onrender.com'; // ← 改成步骤②的真实域名
const CODA_ACCESS_TOKEN = '';                              // ← 改成步骤②填的 token
  2. 把 CODA_CLOUD_API 改成 Render 给的真实域名（结尾不要带斜杠）
  3. 把 CODA_ACCESS_TOKEN 改成步骤②里 CODA_ACCESS_TOKEN 的同一串值
  4. 保存

4 出 Windows / Mac 安装包

桌宠用 electron-builder，打包脚本现成。在 coda-desktop 目录执行

cd /Users/diaoye/Desktop/coda-desktop
npm install            # 首次需要，装 electron 等

# Mac 安装包（.dmg）—— 在 Mac 上直接出
npm run build:mac

# Windows 安装包（.exe）—— Mac 上也能交叉构建
npm run build:win

# 产物在 dist/ 目录
若 Win 包在 Mac 上构建失败：多半是 sidecar 二进制或签名问题。兜底办法：找队友的 Windows 机器跑 npm run build:win，或先只交 Mac 包 + 录屏，路演当天用 Mac 演示。
验证：本机装一次产出的包，双击打开桌宠，点开工作台→扫盘，确认能连云端、出「医疗 88%」+ AI 评审。务必在干净状态下验一遍，模拟评委首次使用。

5 上传安装包 + 写下载链接

  1. 把 dist/ 里的 .dmg 和 .exe 传到 GitHub Release（仓库 → Releases → Draft new release → 拖文件上传），或网盘
  2. 把下载链接写进提交说明 + README.md 的「快速直达」区
  3. 提交物清单里「Demo 公网可访问链接」= 安装包下载链接（+ 后端 /health 作为在线证明）
